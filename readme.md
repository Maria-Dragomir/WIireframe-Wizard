# Wireframe Wizard

## Group Project: Interactive Web Application using a 3rd Party API

## Challenge

Wireframe Wizard is a Week 5 group project created by **Hackers of the Apocalypse**.

Our mission: help users turn rough webpage ideas into a useful starter scaffold without needing to stare blankly at an empty code editor.

The application is designed to:

- Accept and validate user input
- Use Gemini as the API-powered assistant
- Generate a usable HTML and CSS scaffold
- Display results clearly for the user
- Be user-friendly, accessible and responsive
- Be deployed using GitHub Pages

---

## Project Overview

Wireframe Wizard helps users describe a webpage idea and receive starter HTML and CSS that they can build from.

It is not designed to replace developers. It is designed to give beginners a practical starting point, reduce overwhelm, and make the first version of a webpage easier to begin.

---

## Key Features

- Designed to offer the most usable scaffold possible
- Designed to be user-friendly and accessible
- Powered by Gemini
- Generates starter HTML and CSS from user input
- Clear layout and simple interaction
- Responsive design
- Built with HTML, CSS and JavaScript

---

## User Story

As a user, I want to describe a rough webpage idea so that I can receive starter HTML and CSS to help me turn that idea into a real page.
